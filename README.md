## الميزات

يحتوي هذا التطبيق على الميزات الأساسية المتوقعة في تطبيق تواصل اجتماعي:

*   **تسجيل الدخول/التسجيل:** إدارة حسابات المستخدمين.
*   **خلاصة الأخبار (Feed):** عرض منشورات الأصدقاء والصفحات المتابعة.
*   **المنشورات:** إنشاء، تعديل، وحذف المنشورات النصية أو بالصور.
*   **التفاعلات (Reactions):** الإعجاب، الحب، الغضب، وغيرها على المنشورات.
*   **التعليقات:** إضافة والرد على التعليقات.
*   **صفحة الملف الشخصي:** عرض المعلومات الشخصية والمنشورات الخاصة بالمستخدم.
*   **الرسائل (Chat):** نظام مراسلة فوري بين المستخدمين (اختياري).
*   **الإشعارات:** تنبيه المستخدم بالتفاعلات الجديدة والرسائل.

## التثبيت

### المتطلبات الأساسية

تأكد من تثبيت الأدوات التالية على نظامك:

*   Node.js (يفضل الإصدار 14+)
*   npm أو yarn
*   Git

### الإعداد

1.  **استنساخ المستودع (Clone the repository):**
        git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
    cd YOUR_REPO_NAME
    
2.  **تثبيت التبعيات (Install dependencies):**
        npm install
    # أو
    yarn install
    
3.  **إعداد متغيرات البيئة:**
    أنشئ ملف `.env` في الجذر واملأه بالمتغيرات اللازمة (مثل مفاتيح API، وسلسلة اتصال قاعدة البيانات):
        # مثال للمتغيرات (قد تختلف حسب التقنية المستخدمة)
    PORT=3000
    DATABASE_URL="mongodb://localhost:27017/facebook_clone_db"
    JWT_SECRET="YOUR_SUPER_SECRET_KEY"
    
4.  **تشغيل التطبيق:**
        npm start
    # أو لتشغيل وضع التطوير
    npm run dev
    
سيصبح التطبيق متاحًا على العنوان `http://localhost:3000` (أو المنفذ الذي حددته).

## الاستخدام

### المستخدمون

1.  قم بزيارة صفحة التسجيل لإنشاء حساب جديد.
2.  قم بتسجيل الدخول باستخدام بيانات اعتمادك.
3.  يمكنك تصفح خلاصة الأخبار، إنشاء منشورك الأول، أو البحث عن أصدقاء.

### API (للمطورين)

إذا كان التطبيق يقدم واجهة برمجية، يمكنك استخدام endpoints التالية:

| المسار (Path) | الطريقة (Method) | الوصف |
| :--- | :--- | :--- |
| `/api/v1/auth/register` | `POST` | تسجيل مستخدم جديد |
| `/api/v1/auth/login` | `POST` | تسجيل الدخول والحصول على رمز JWT |
| `/api/v1/posts` | `GET` | جلب جميع المنشورات (الخلاصة) |
| `/api/v1/posts` | `POST` | إنشاء منشور جديد |

راجع مجلد `docs/API.md` (إذا وُجد) للحصول على توثيق كامل للواجهة البرمجية.
